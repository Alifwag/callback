Information Center

  
    
    
  • Callback: Data sent back after an event occurs.
    • 
    
  • Webhook: An automated callback triggered by API events.
    • 
    
  • Supported Methods: POST, PUT, PATCH.
    • 
    
  • Headers: Metadata such as Content-Type, Authorization.
    • 
    
  • Payload: JSON with event details.
    • 
  






  
FAQ / Questions & Answers

  

    
How do I use this tool?

    
Fill in endpoint, method, headers, and payload. Click send. Check response below.

  

  

    
Can I test without a server?

    
Yes. Use the simulate button to preview payload without sending any request.

  

  

    
Is my data stored?

    
Only locally in your browser’s localStorage, not on any external server.

  

  

    
Can I send files?

    
This demo focuses on JSON payloads. For files, use multipart/form-data and adjust code accordingly.

  

  

    
What about authentication?

    
Add headers like Authorization: Bearer <token> in the headers section.

Interactive Tutorial

  
Step 1: Enter your endpoint URL in the input box.

  
Step 2: Choose HTTP method (POST is most common).

  
Step 3: Edit headers if needed (JSON format).

  
Step 4: Edit payload (JSON) or leave default.

  
Step 5: Click “Send Callback” to send. Observe response.

  
Step 6: Check the history section to review what you sent.

  
Tutorial finished! You can restart anytime.
